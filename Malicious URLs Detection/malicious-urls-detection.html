Project overview¶

The project consisted to evaluate different classification models to predict malicious and benign websites, based on application layer and network characteristics.

Problem statement¶

Confidentiality and Privacy is the biggest challenge faced. If a user access the

malicious website through a search engine with no perception, malicious scripts

usually launch attacks to install rogue program, steal personal identities and

credentials, so our Problem is to develop an algorithm which accurately predicts

the malicious and the benign URLs.

This is a binary classification supervised problem that takes the URL as an input and returns

whether it contains malicious content or not as an output.

We've got our dataset from Kaggle platform: https://www.kaggle.com/xwolf12/malicious-and-benign-websites

  • Data set characteristics: Multivariate, Time-series
  • Attribute characteristics: Integer, real, characters
  • Associated tasks: Classification
  • Number of instances: 1782
  • Number of attributes: 21
  • Data format: csv

The data is imbalanced and contains 1782 labeled URL:

  • 216 malicious URLs (12 %)

in This project I'll evaluate different supervised algorithms to classify the URLs based on this data and the features related to each URL like URL length and the number of special characters in the URL.

Helper functions¶

  • print_score
    • takes the classifier and print the score.
    • if the data is imbalanced the score will contain only the recall metric.
In [79]:
from sklearn.model_selection import cross_val_predict, cross_val_score
from sklearn.metrics import confusion_matrix,accuracy_score, recall_score

# Visualize our results
def print_score(classifier, imbalanced_data = False):
    if imbalanced_data == True:
        train_score = recall_score(y_train_orig,classifier.predict(X_train_orig))
        test_score = recall_score(y_test_orig,classifier.predict(X_test_orig))
        print("Imbalanced data:\n")
        print("Training results:\n")
        print('Recall Score: {0:.4f}\n'.format(train_score))
        print("Testing results:\n")
        print('Recall Score: {0:.4f}\n'.format(test_score))
        print('Confusion Matrix:\n{}\n'.format(confusion_matrix(y_test,classifier.predict(X_test))))
        return {'training Recall':train_score,'testing Recall':test_score}
    train_score = accuracy_score(y_train,classifier.predict(X_train))
    train_recall = recall_score(y_train,classifier.predict(X_train))
    test_score = accuracy_score(y_test,classifier.predict(X_test))
    test_recall = recall_score(y_test,classifier.predict(X_test))
    print("Balanced data:\n")
    print("Training results:\n")
    print('Accuracy Score: {0:.4f}\n'.format(train_score))
    print("Testing results:\n")
    print('Accuracy Score: {0:.4f}\n'.format(test_score))
    print('Confusion Matrix:\n{}\n'.format(confusion_matrix(y_test,classifier.predict(X_test))))
    return {'training Recall':train_recall,'training accuracy':train_score,'testing Recall':test_recall,'testing accuracy':test_score}
